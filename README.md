# SORTING VISUALISER
Deep dive into understanding the working of different sorting algorithms by viusalising their working!

Currently, the application incudes 5 sorting algorithms:

* INSERTION SORT
* BUBBLE SORT
* SELECTION SORT
* MERGE SORT
* QUICK SORT (both recursive and iterative)
